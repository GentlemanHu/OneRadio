## OneRadio 收音机

---
#### 整体架构

<pre><code class="java">
    |- pers.hu.oneradio
            |- activities   //活动或UI包
            |- net          //网络处理相关包
            |- utils        //工具类包
            <s>|- controllers  //控制器包(暂不需要)</s>
    |- MainActivity.java    //入口


    ------------------
    |- res
        |- layout //UI布局
        |- values //变量注册信息

</code></pre>
> 说明：UI布局不需要创建Activity，只需要在layout中创建布局信息，搞好就行
> 如果layout中布局信息的组件<自定义命名>，请在values中对应变量中注册添加
